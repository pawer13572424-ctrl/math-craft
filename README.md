# ⛏️ さんすうクラフト - MATH CRAFT

マイクラ風UIの小学1年生向け計算アプリです。クリーパーがいっぱい！

## ✨ 特徴

- **🎮 ぼうけんモード** — 問題を解いてレベルアップ！8段階の難易度が自動で上がっていく
- **📝 テストモード** — 5・10・20問から選べるテスト。結果と正答率を表示
- **🔊 サウンド** — 正解時にXPオーブ風チャイム、不正解時にダメージ音、レベルアップ時にファンファーレ
- **💚 クリーパーだらけ** — ピクセルアートのクリーパーが画面中に！タップするとシューッと鳴く
- **📱 モバイル対応** — タッチ用の数字パッド付き

## 🚀 セットアップ

```bash
# クローン
git clone https://github.com/<your-username>/math-craft.git
cd math-craft

# インストール
npm install

# 開発サーバー起動
npm run dev
```

ブラウザで `http://localhost:5173` を開くと遊べます。

## 🌐 GitHub Pages にデプロイ

1. `package.json` の `homepage` を自分のURLに書き換える
2. `vite.config.js` の `base` も必要に応じて変更

```bash
npm run deploy
```

3. GitHubのリポジトリ Settings → Pages → Source を `gh-pages` ブランチに設定
